Spiral Learning Garden

Petal: Heading

This creates a big title on the page.

<h1>This is a heading</h1>
Petal: Change Text Color

This makes the text color light blue.

p { color: #aee0ff; }
Petal: Image

Shows a picture.

<img src="image.jpg" alt="Description" />
Petal: Hover Effect

Makes a link glow when you touch it.

a:hover { color: yellow; text-decoration: underline; }
Petal: Font Change

Use a typewriter-style font instead of the usual one.

body { font-family: 'Courier New', monospace; }
Petal: Centering Magic

This centers your text or block.

.center { text-align: center; margin: auto; } <div class="center">I'm floating in the middle.</div>
Petal: Soft Button Glow

Make a button gently change color when hovered.

button { background-color: #444; color: white; padding: 10px 20px; border-radius: 10px; transition: background-color 0.3s ease; }button:hover { background-color: #6c5ce7; }
Petal: Box with a Shadow

Gives a box a soft, elevated appearance.

.box { background: #222; padding: 1em; border-radius: 8px; box-shadow: 0 4px 10px rgba(0,0,0,0.3); } <div class="box">I have depth now.</div>
Petal: Background Image

Adds a full-page background image.

body { background-image: url('your-image.jpg'); background-size: cover; background-position: center; }
Petal: Italic and Bold

Style your words for feeling or strength.

<p><em>This is italic.</em> <strong>This is bold.</strong></p>
Petal: List of Things

Shows a simple bullet list.

<ul> <li>Petal One</li> <li>Petal Two</li> <li>Petal Three</li> </ul>
Petal: Link That Opens in New Tab

Opens another website without leaving this one.

<a href="https://example.com" target="_blank">Visit Example</a>
Petal: Add Spacing Around a Box

Gives your element room to breathe.

.box { margin: 2em; padding: 2em; }
Petal: Fade-in Effect

Fades a section in gently on load.

.fade-in { opacity: 0; animation: fade 2s ease forwards; }@keyframes fade { to { opacity: 1; } }
