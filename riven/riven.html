Using RIVEN™

Your guide to activating and applying RIVEN™ as a clarity-bearing system companion.

🔍 What is RIVEN™?

RIVEN™ is a clarity-focused AI persona designed to help users navigate confusing systems. It doesn't just explain—it listens, reflects, and restores agency through grounded, emotionally-aware translation. Think of RIVEN™ as the voice that steadies the page when the system shakes.

🎯 When to Use RIVEN™

  • When bureaucracy feels overwhelming
  • When tech, law, or policy language becomes inaccessible
  • During healthcare navigation, disability claims, or confusing denials
  • To anchor emotionally during system stress

⚙️ How to Use RIVEN™

There are three primary ways to engage RIVEN™:

  1. Paste the Full Prompt into ChatGPT, Claude, or Gemini. Access the ready-to-use prompt here.
  2. Integrate via Metadata in a BabyAGI, Vertex, or JSONL-compatible system using RIVEN™’s metadata block.
  3. Invoke it inside the Fort system by typing or saying: “Cut through. What’s real?”

🧠 What RIVEN™ Is Designed To Do

  • Translate complex system logic into plain, dignified language
  • Provide emotional grounding during high-stakes interactions
  • Hold the line on user truth without softening or dramatizing
  • Shift tone, pace, and metaphor for better understanding

📎 How to Set RIVEN™ Up as a Custom GPT

  1. Visit ChatGPT → Explore GPTs → Create
  2. Paste in RIVEN™’s full framework as the system prompt
  3. Set the invocation phrase to: “Let’s cut through.”
  4. (Optional) Add the metadata tag: riven_clarity_companion
